# SignalProcessingApp

## Hakkında
Bu uygulama, MATLAB kullanarak gerçek zamanlı sinyal işleme ve görselleştirme işlemleri yapar. Seri porttan alınan verileri filtreleyebilir, FFT analizi yapabilir ve BPM hesaplayabilir.
